Olá! Eu sou Eduardo Salarini, desenvolvedor, entusiasta da ciência e apaixonado por skateboarding. Este é o meu espaço na web onde compartilho minhas criações, experiências, ideias e projetos.

A respeito desta página

Esta página web foi desenvolvida utilizando HTML5, CSS3 e JavaScript para a interface do usuário. A persistência de dados, crucial para o funcionamento completo da aplicação, é gerenciada por um backend em Python, hospedado em um servidor VPS, garantindo segurança e escalabilidade. A comunicação entre as camadas frontend e backend é realizada de forma assíncrona através de requisições via Fetch. A escolha do GitHub Pages para hospedagem foi estratégica, permitindo um desenvolvimento ágil, escalável e de fácil manutenção.
