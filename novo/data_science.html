Projetos de Data Science

Detecção de Fraudes em Transações Financeiras

Descrição: Construir um modelo de classificação para detectar transações suspeitas com base em padrões de comportamento.

Tech stack: Python, scikit-learn, PyCaret, SMOTE.

Diferencial: Implementar um sistema que atualiza o modelo com novos dados (aprendizado contínuo).

Previsão de Demanda para E-commerce

Descrição: Usar séries temporais para prever a demanda de produtos em um e-commerce, ajudando na gestão de estoque.

Tech stack: Python, statsmodels, Prophet, TensorFlow (LSTMs).

Diferencial: Criar um sistema que sugere compras automáticas para otimizar estoques.

Identificação de Espécies de Plantas ou Animais

Descrição: Criar um modelo que identifica espécies de plantas ou animais a partir de imagens, auxiliando biólogos e ambientalistas.

Tech stack: Python, OpenCV, TensorFlow, PlantNet, iNaturalist.

Diferencial: Poderia ser usado como um app de campo para cientistas e amantes da natureza.

Detecção Automática de Meteoritos

Descrição: Criar um sistema que analisa imagens do céu em tempo real e detecta meteoritos com visão computacional.

Tech stack: Python, OpenCV, TensorFlow, bases da NASA e IMO.

Diferencial: Poderia contribuir com a ciência cidadã e monitoramento de impactos.

Predição de Erupções Solares

Descrição: Criar um modelo que prevê erupções solares usando dados de telescópios espaciais.

Tech stack: Python, TensorFlow, dados da NASA (Solar Dynamics Observatory).

Diferencial: Poderia ajudar a prever eventos que afetam satélites e redes elétricas na Terra.

Análise de Radiação Cósmica e seus Efeitos na Terra

Descrição: Criar um modelo que analisa o impacto da radiação cósmica na saúde humana e nos sistemas eletrônicos.

Tech stack: Python, Pandas, TensorFlow, bases como ESA e NOAA.

Diferencial: Poderia ser útil para a indústria aeroespacial e telecomunicações.

Algoritmo de Previsão de Epidemias de Alergias Sazonais

Descrição: Criar um sistema que prevê surtos de alergias sazonais com base em dados climáticos e níveis de pólen.

Tech stack: Python, Pandas, scikit-learn, dados de redes meteorológicas.

Diferencial: Pode ajudar pessoas alérgicas a se prepararem melhor.
